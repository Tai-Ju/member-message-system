Features
功能特色
🔐
使用者註冊
信箱重複驗證機制，確保每個信箱只能註冊一次帳號。
🔑
安全登入
Session 狀態管理，使用 Secret Key 加密，登入後自動導向會員專區。
💬
即時留言
登入會員可發表留言，按時間倒序排列，即時顯示最新訊息。
🛡️
路由保護
未登入用戶自動重導向至登入頁，保護會員專屬功能。
🔒
環境變數管理
敏感資訊透過 .env 與程式碼分離，.env 不上傳至 GitHub。
🚪
登出功能
清除 Session 資料，安全登出並返回登入頁面。

API Routes
路由對照表
Method路徑說明狀態
GET/顯示首頁（登入 / 註冊表單）公開
POST/signup註冊新帳戶（name, email, password）公開
POST/signin會員登入驗證，建立 Session公開
GET/error顯示錯誤訊息頁面公開
GET/member會員專區（含留言列表）需登入
POST/message發表留言（content）需登入
GET/signout登出，清除 Session需登入

Install
本地運行
01
Clone 專案
git clone https://github.com/Tai-Ju/member-message-system.git
cd member-message-system
02
安裝依賴套件
npm install
03
設定環境變數
複製 .env.example 為 .env，填入 MongoDB 連線字串與 Session Secret
# Windows
copy .env.example .env

MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/
SESSION_SECRET=your-random-secret-key
PORT=3000
04
啟動伺服器
node index.js
